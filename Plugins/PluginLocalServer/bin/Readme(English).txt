Plugin installation instructions:
1. If your current Windows version has a version number lower than 3.2, double-click ftnninstall.exe to install it.
2. pluginserver.dll is a compiled version that has been compiled successfully. If necessary, compile it yourself.
3. Copy config.ini, pluginserver.dll to the plugin subdirectory under the ftnn installation directory.
4. python/ftnn_api_test.py is a sample python call, it is recommended to run in python2.7 and above.
